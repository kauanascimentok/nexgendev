# NexGen Dev — Site de Vendas

Site profissional para agência de desenvolvimento web jovem.

## Estrutura de Arquivos

```
nexgendev/
├── index.html          ← Página principal (abra no navegador)
├── css/
│   └── style.css       ← Todos os estilos
├── js/
│   └── main.js         ← Interações e animações
└── README.md           ← Este arquivo
```

## Como usar

1. Abra o arquivo `index.html` no seu navegador
2. Para publicar na internet, hospede os arquivos em qualquer servidor (Vercel, Netlify, GitHub Pages — todos gratuitos)

## O que personalizar

### Informações de contato (index.html)
- **WhatsApp**: Troque `5592999999999` pelo número real (código do país + DDD + número)
- **E-mail**: Troque `contato@nexgendev.com.br` pelo e-mail real
- **Instagram**: Troque `@nexgendev` pelo @ real
- **Localização** no footer: já está Rio Branco, Acre

### Nome da agência
- Busque "NexGen Dev" no index.html e troque pelo nome real

### Preços (seção #precos)
- Starter: R$ 497
- Profissional: R$ 997
- Premium: R$ 1.997
- Ajuste conforme sua estratégia

### Depoimentos
- Substitua os nomes e textos por depoimentos reais dos seus clientes

### Stats / números
- Hero: 50+, 98%, 3+ anos, 24h
- Sobre: os mesmos números
- Ajuste para refletir a realidade de vocês

## Seções do site

1. **Hero** — Apresentação principal com cards flutuantes e estatísticas
2. **Serviços** — 6 cards com os serviços oferecidos
3. **Sobre** — Sobre a equipe, diferenciais e stack tecnológico
4. **Processo** — 5 etapas de trabalho
5. **Preços** — 3 planos com toggle mensal/anual
6. **Depoimentos** — 6 avaliações de clientes
7. **Contato** — Formulário + canais de comunicação
8. **Footer** — Links e informações da empresa
